Objetivo

Desarrollar una página web explicando las sentencias de control, su sintaxis y conceptos fundamentales para comprender y aplicar estructuras de control en programación.

¿Qué son las Sentencias de Control?

Las sentencias de control son estructuras fundamentales en programación que permiten controlar el flujo de ejecución de un programa. Estas determinan qué código se ejecuta, cuándo se ejecuta y cuántas veces se repite.

Existen dos tipos principales de sentencias de control:

  • Sentencias Condicionales: Permiten ejecutar diferentes bloques de código según se cumplan o no ciertas condiciones (if, else, switch).
  • Sentencias de Repetición (Bucles): Permiten ejecutar un bloque de código múltiples veces (for, while, do-while).

Sentencias Condicionales

1. if / else / else if

La sentencia if evalúa una condición y ejecuta un bloque de código si esta es verdadera. Puede complementarse con else y else if para manejar múltiples casos.

Sintaxis:

if (condición) { // código si la condición es verdadera } else if (otraCondición) { // código si la segunda condición es verdadera } else { // código si ninguna condición anterior es verdadera }

Ejemplo:

let edad = 18; if (edad < 18) { console.log("Eres menor de edad"); } else if (edad === 18) { console.log("Acabas de cumplir la mayoría de edad"); } else { console.log("Eres mayor de edad"); }

2. switch

La sentencia switch evalúa una expresión y ejecuta el código del caso que coincida con el valor de la expresión. Es útil cuando se tienen múltiples valores específicos a comparar.

Sintaxis:

switch (expresión) { case valor1: // código para valor1 break; case valor2: // código para valor2 break; default: // código si ningún caso coincide }

Ejemplo:

let dia = 3; let nombreDia; switch (dia) { case 1: nombreDia = "Lunes"; break; case 2: nombreDia = "Martes"; break; case 3: nombreDia = "Miércoles"; break; default: nombreDia = "Día no válido"; } console.log(nombreDia); // Imprime: Miércoles

Sentencias de Repetición (Bucles)

3. for

El bucle for se utiliza cuando sabemos exactamente cuántas veces queremos repetir un bloque de código. Es ideal para iterar sobre rangos numéricos o elementos de un array.

Sintaxis:

for (inicialización; condición; incremento) { // código a repetir }

Ejemplo:

// Imprimir números del 1 al 5 for (let i = 1; i <= 5; i++) { console.log("Número: " + i); } // Salida: // Número: 1 // Número: 2 // Número: 3 // Número: 4 // Número: 5

4. while

El bucle while ejecuta un bloque de código mientras una condición sea verdadera. La condición se evalúa antes de cada iteración.

Sintaxis:

while (condición) { // código a repetir mientras la condición sea verdadera }

Ejemplo:

let contador = 1; while (contador <= 3) { console.log("Iteración: " + contador); contador++; } // Salida: // Iteración: 1 // Iteración: 2 // Iteración: 3

5. do-while

El bucle do-while es similar al while, pero garantiza que el código se ejecute al menos una vez, ya que la condición se evalúa después de cada iteración.

Sintaxis:

do { // código a repetir } while (condición);

Ejemplo:

let numero = 0; do { console.log("El número es: " + numero); numero++; } while (numero < 3); // Salida: // El número es: 0 // El número es: 1 // El número es: 2
