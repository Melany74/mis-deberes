1. ¿Qué es una Expresión?

Una expresión en programación es cualquier combinación válida de valores, variables, operadores y funciones que puede ser evaluada para producir un resultado.

Concepto clave: Las expresiones son los "cálculos" del código. Siempre producen un valor como resultado.

Tipos de Expresiones

1️.- Expresiones Aritméticas

Realizan operaciones matemáticas y producen un valor numérico.

2️.- Expresiones Lógicas

Evalúan condiciones y producen valores booleanos (true o false).

3️.- Expresiones de Cadena

Manipulan texto y producen valores de tipo string.

4️.- Expresiones de Asignación

Asignan valores a variables y retornan el valor asignado.

Ejemplos de Expresiones

Expresiones Aritméticas:

// Expresión simple 5 + 3 // Resultado: 8 // Expresión con variables let x = 10; let y = 20; x * y - 50 // Resultado: 150 // Expresión compleja (x + y) / 2 // Resultado: 15 (promedio)

Expresiones Lógicas:

// Comparaciones 10 > 5 // Resultado: true 3 === 3 // Resultado: true // Operadores lógicos (5 > 3) && (10 < 20) // Resultado: true (5 === 3) || (7 > 2) // Resultado: true

Expresiones de Cadena:

// Concatenación "Hola" + " " + "Mundo" // Resultado: "Hola Mundo" // Template literals let nombre = "Ana"; `Hola, ${nombre}!` // Resultado: "Hola, Ana!"

Expresiones de Asignación:

// Asignación simple let edad = 25; // Asigna 25 a edad // Asignación con operación edad += 1; // edad = edad + 1 edad *= 2; // edad = edad * 2

2. Tipos de Operadores

Los operadores son símbolos que le indican al programa que realice operaciones específicas sobre uno o más valores.

Operadores Aritméticos

Realizan operaciones matemáticas básicas.

Operador Nombre Ejemplo Resultado
+ Suma 5 + 3 8
- Resta 10 - 4 6
* Multiplicación 3 * 4 12
/ División 20 / 5 4
% Módulo (residuo) 10 % 3 1
** Exponenciación 2 ** 3 8
++ Incremento x++ x + 1
-- Decremento x-- x - 1

Operadores Relacionales (Comparación)

Comparan dos valores y devuelven true o false.

Operador Nombre Ejemplo Resultado
== Igual a (valor) 5 == "5" true
=== Estrictamente igual (valor y tipo) 5 === "5" false
!= Diferente de 5 != 3 true
!== Estrictamente diferente 5 !== "5" true
> Mayor que 10 > 5 true
< Menor que 3 < 8 true
>= Mayor o igual que 5 >= 5 true
<= Menor o igual que 3 <= 7 true

Operadores Lógicos

Combinan o invierten valores booleanos.

Operador Nombre Descripción Ejemplo
&& AND (Y) Verdadero si ambas condiciones son verdaderas (5 > 3) && (10 > 7) → true
|| OR (O) Verdadero si al menos una condición es verdadera (5 < 3) || (10 > 7) → true
! NOT (NO) Invierte el valor booleano !(5 > 3) → false

Operadores de Asignación

Asignan valores a variables.

Operador Ejemplo Equivalente a
= x = 5 x = 5
+= x += 3 x = x + 3
-= x -= 2 x = x - 2
*= x *= 4 x = x * 4
/= x /= 2 x = x / 2
%= x %= 3 x = x % 3

Ejemplo combinado de operadores:

let a = 10; let b = 5; let c = 3; // Operadores aritméticos let suma = a + b; // 15 let producto = a * c; // 30 // Operadores relacionales let esMayor = a > b; // true let sonIguales = a === b; // false // Operadores lógicos let resultado = (a > b) && (c < b); // true // Operadores de asignación a += 5; // a ahora es 15 b *= 2; // b ahora es 10

3. Ejercicios sobre Funciones

Ejercicio 1

Crear una función que calcule el área de un rectángulo

Problema: Escribe una función que reciba la base y altura de un rectángulo y retorne su área.

Solución Paso a Paso

Paso 1: Definir la función

Creamos una función con dos parámetros: base y altura

Paso 2: Calcular el área

El área de un rectángulo es: base × altura

Paso 3: Retornar el resultado

Usamos return para devolver el valor calculado

// Solución function calcularAreaRectangulo(base, altura) { let area = base * altura; return area; } // Forma simplificada const calcularArea = (base, altura) => base * altura; // Uso let resultado = calcularAreaRectangulo(5, 3); console.log(`El área es: ${resultado}`); // El área es: 15
Ejercicio 2

Crear una función que determine si un número es par o impar

Problema: Escribe una función que reciba un número y retorne "Par" si es par, o "Impar" si es impar.

Solución Paso a Paso

Paso 1: Definir la función

Creamos una función que reciba un número como parámetro

Paso 2: Usar el operador módulo (%)

Si numero % 2 === 0, el número es par

Paso 3: Retornar el resultado

Usamos un condicional para retornar "Par" o "Impar"

// Solución function esParOImpar(numero) { if (numero % 2 === 0) { return "Par"; } else { return "Impar"; } } // Forma simplificada con operador ternario const parImpar = (num) => (num % 2 === 0) ? "Par" : "Impar"; // Uso console.log(esParOImpar(8)); // "Par" console.log(esParOImpar(7)); // "Impar"
Ejercicio 3

Crear una función que convierta grados Celsius a Fahrenheit

Problema: Escribe una función que convierta una temperatura de Celsius a Fahrenheit usando la fórmula: F = (C × 9/5) + 32

Solución Paso a Paso

Paso 1: Definir la función

Creamos una función que reciba los grados Celsius

Paso 2: Aplicar la fórmula

F = (C × 9/5) + 32

Paso 3: Retornar el resultado

Devolvemos la temperatura en Fahrenheit

// Solución function celsiusAFahrenheit(celsius) { let fahrenheit = (celsius * 9 / 5) + 32; return fahrenheit; } // Forma simplificada const convertirTemp = (c) => (c * 9 / 5) + 32; // Uso console.log(celsiusAFahrenheit(0)); // 32°F console.log(celsiusAFahrenheit(100)); // 212°F console.log(celsiusAFahrenheit(25)); // 77°F
Ejercicio 4

Crear una función que encuentre el número mayor de tres números

Problema: Escribe una función que reciba tres números y retorne el mayor de ellos.

Solución Paso a Paso

Paso 1: Definir la función

Creamos una función con tres parámetros

Paso 2: Comparar los números

Usamos condicionales para comparar los tres números

Paso 3: Retornar el mayor

Devolvemos el número que sea mayor que los otros dos

// Solución con condicionales function encontrarMayor(num1, num2, num3) { if (num1 >= num2 && num1 >= num3) { return num1; } else if (num2 >= num1 && num2 >= num3) { return num2; } else { return num3; } } // Solución usando Math.max() const numeroMayor = (a, b, c) => Math.max(a, b, c); // Uso console.log(encontrarMayor(5, 12, 8)); // 12 console.log(encontrarMayor(15, 3, 20)); // 20 console.log(numeroMayor(7, 9, 4)); // 9
Ejercicio 5

Crear una función que calcule el promedio de un array de números

Problema: Escribe una función que reciba un array de números y retorne el promedio de todos ellos.

Solución Paso a Paso

Paso 1: Definir la función

Creamos una función que reciba un array como parámetro

Paso 2: Sumar todos los elementos

Usamos un bucle o el método reduce() para sumar

Paso 3: Dividir entre la cantidad

Dividimos la suma entre el número de elementos

Paso 4: Manejar casos especiales

Verificar que el array no esté vacío para evitar dividir entre 0

// Solución con bucle for function calcularPromedio(numeros) { if (numeros.length === 0) { return 0; // Array vacío } let suma = 0; for (let i = 0; i < numeros.length; i++) { suma += numeros[i]; } let promedio = suma / numeros.length; return promedio; } // Solución con reduce() const promedioArray = (arr) => { if (arr.length === 0) return 0; const suma = arr.reduce((acc, num) => acc + num, 0); return suma / arr.length; }; // Uso let notas = [85, 90, 78, 92, 88]; console.log(calcularPromedio(notas)); // 86.6 let edades = [20, 25, 30]; console.log(promedioArray(edades)); // 25
