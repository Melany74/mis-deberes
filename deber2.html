¿Qué son las Funciones en Programación?

Las funciones son bloques de código reutilizables que realizan una tarea específica. Son fundamentales en programación porque permiten:

  • Reutilización de código: Escribes el código una vez y lo usas múltiples veces
  • Organización: Divides programas grandes en partes más pequeñas y manejables
  • Mantenibilidad: Facilita la corrección de errores y actualización del código
  • Abstracción: Oculta la complejidad interna y expone solo lo necesario

Concepto clave: Una función es como una "máquina" que recibe datos de entrada (parámetros), realiza una operación, y produce un resultado (valor de retorno).

Tipos de Funciones en JavaScript

1. Función Declarada (Function Declaration)

Es la forma tradicional de declarar funciones. Se define usando la palabra clave function.

Sintaxis:

function nombreFuncion(parametro1, parametro2) { // código a ejecutar return resultado; }

Ejemplo:

function sumar(a, b) { return a + b; } // Llamar a la función let resultado = sumar(5, 3); console.log(resultado); // Imprime: 8

2. Expresión de Función (Function Expression)

La función se asigna a una variable. No puede ser llamada antes de su definición.

Ejemplo:

const multiplicar = function(a, b) { return a * b; }; let resultado = multiplicar(4, 5); console.log(resultado); // Imprime: 20

3. Función Flecha (Arrow Function)

Sintaxis moderna y más concisa introducida en ES6. Ideal para funciones cortas.

Sintaxis:

const nombreFuncion = (parametros) => { // código return resultado; }; // Si solo tiene una expresión, puede ser más corta: const nombreFuncion = (parametros) => resultado;

Ejemplos:

// Función flecha básica const restar = (a, b) => { return a - b; }; // Función flecha simplificada (una línea) const dividir = (a, b) => a / b; // Función flecha con un solo parámetro (sin paréntesis) const cuadrado = x => x * x; console.log(cuadrado(5)); // Imprime: 25

4. Función Anónima

Funciones sin nombre, generalmente usadas como argumentos de otras funciones.

Ejemplo:

// Función anónima como callback setTimeout(function() { console.log("Esto se ejecuta después de 2 segundos"); }, 2000); // O con función flecha setTimeout(() => { console.log("Versión con arrow function"); }, 2000);

Comparación de Tipos de Funciones

Tipo Características Cuándo usar
Declarada Puede llamarse antes de ser definida (hoisting) Funciones principales del programa
Expresión Solo disponible después de su definición Cuando necesitas asignar funciones a variables
Flecha Sintaxis corta, no tiene su propio "this" Callbacks, funciones cortas, métodos de arrays
Anónima Sin nombre, usada como argumento Callbacks de una sola vez, event listeners

Parámetros y Valores de Retorno

Parámetros

Los parámetros son variables que recibe una función para trabajar con ellas.

Ejemplo con múltiples parámetros:

function saludar(nombre, edad) { console.log(`Hola, me llamo ${nombre} y tengo ${edad} años`); } saludar("María", 20); // Imprime: Hola, me llamo María y tengo 20 años

Valores de Retorno

El return devuelve un valor al lugar donde se llamó la función.

Ejemplo:

function calcularArea(base, altura) { let area = base * altura; return area; } let resultado = calcularArea(5, 10); console.log("El área es: " + resultado); // El área es: 50
